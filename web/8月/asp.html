ASP

ASP是什么？

asp是指 Active Server Pages （动态服务器页面）

在ASP中使用JavaScript

必须在页面的顶端插入一行语言设定：

			<% @ language="javascript" %>

ASP语法

向浏览器输出：

		第一种方法:<% response.write("输入的文本或值") %>
		第二种方法:<% ="输入文本或值" %>

声明变量

		<%
		    dim name
		    name="Donald Duck"
		    response.write("My name is: " & name)
		%>

		运行结果：My name is: Donald Duck

声明数组

		<%
		    Dim fname(2),i
		    fname(0) = "George"
		    fname(1) = "John"
		    fname(2) = "Thomas"

		    For i = 0 to 2
		          response.write(fname(i) & "<br />")
		    Next
		%>

		运行结果：
			George
			John
			Thomas

循环生成HTML标题

		<%
		    dim i 
		    for i=1 to 6
		       response.write("Header " & i & "</h" & i & ">")
		    next
		%>	

		运行结果：
		<h1>Header</h1>
		<h2>Header</h2>
		<h3>Header</h3>
		<h4>Header</h4>
		<h5>Header</h5>
		<h6>Header</h6>

ASP程序

调用使用JavaScript的子程序

		
		<% @ language="javascript" %>

		<html>
		<head>

		<%
		function jsproc(num1,num2)
		{
		Response.Write(num1*num2)
		}
		%>

		</head>
		<body>

		<p>结果：<%jsproc(3,4)%></p>

		</body>
		</html>

		运行结果：
		    结果：12

调用使用JavaScript的子程序

		<html>
		<head>

		<script  language="javascript" runat="server">
		function jsproc(num1,num2)
		{
		Response.Write(num1*num2)
		}
		</script>

		</head>
		<body>

		<p>结果：<%call jsproc(3,4)%></p>

		</body>
		</html>

		运行结果：
		    结果：12

ASP表单和用户输入

使用 method="get" 的表单

		<form action="/example/aspe/demo_aspe_reqquery.asp" method="get">
		    您的姓名：<input type="text" name="fname" size="20" />
		    <input type="submit" value="提交" />
		</form>

		<%
		    dim fname
		    fname=Request.QueryString("fname")
		    If fname<>"" Then
		          Response.Write("你好！" & fname & "！<br />")
		          Response.Write("今天过得怎么样？")
		    End If
		%>

		运行结果：
			你好！‘输入框的值’！
			今天过得怎么样？
