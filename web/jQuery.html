jQuery

获得内容 - text()、html() 以及 val()

三个简单实用的用于 DOM 操作的 jQuery 方法:

  • text() - 设置或返回所选元素的文本内容
    • 			$("#点击事件的ID").click(function(){
			    alert("Text: " + $("#显示的标签ID").text());
			});
  • html() - 设置或返回所选元素的内容（包括 HTML 标记）
    • 			$("#点击事件的ID").click(function(){
			  alert("HTML: " + $("#显示的标签ID").html());
			});
  • val() - 设置或返回表单字段的值
    • 			$("#点击事件的ID").click(function(){
			  alert("Value: " + $("#input的ID").val());
			});
  • 获取属性 - attr()
    • 			$("点击事件的ID").click(function(){
			  alert($("#标签ID").attr("标签的属性名称"));
			});
