XML

  • XML是一种标记语言，很类似HTML

  • XML的设计宗旨是传输数据，而非显示数据

  • XML标签没有被预定义。您需要自行定义标签。

  • XML被设计为具有自我描述性。

  • XML是W3C的推荐标准

XML是独立于软件和硬件的信息传输工具。 目前，XML在Web中起到的作用不会亚于一直作为 Web 基石的 HTML。 XML无所不在。XML是各种应用程序之间进行数据传输的最常用的工具，并且在信息存储和描述领域变得越来越流行。
